King of Legends — напряжённый 2D аркадный бой в реальном времени!
Сражайся один на один с друзьями по локальной сети или Wi-Fi.
Атакуй противника, уклоняйся и стань королём легенд!
Поддержка ПК и Android. Создавай сервер или подключайся к другу — битва начинается за секунды!
Для тестирования необходимо ДВА устройства подключенных к одной сети WI-FI возможно (с пк и телефона)
На одном устройстве необходимо запустить сервер (кнопка в меню HOST GAME)
На втором необходимо ввести IP адрес первого устройства выбрав в меню JOIN GAME
Для того чтобы спрыгнуть с платформы необходимо удерживать СТИК в направлении ВНИЗ
